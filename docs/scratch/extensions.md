## 删除

还有 11 个扩展类，可以通过选择添加到阻止列表中，以及 3 个在其他版本的 Scratch 上不可用的 Raspberry Pi 扩展。块类别显示在块调色板左侧的类别中，添加扩展时将显示在那里。
